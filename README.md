# Bid Compare

Bid Compare is a Streamlit-based tool for analyzing and comparing construction bid data from multiple CSVs. It enables year-over-year comparisons, detailed line item analysis, and contractor/unit price breakdownsâ€”all with a clean, filter-driven sidebar interface.

## Features

- Multi-year bid comparison
- Detailed line item analysis
- Contractor performance tracking
- Unit price breakdown
- Interactive filtering

## Usage

1. Place your bid CSV files in the project directory
2. Run the Streamlit app:

   ```bash
   streamlit run main.py
   ```

3. Use the sidebar to select:
   - CSV files to compare
   - Contractors to analyze
   - Sections to examine
